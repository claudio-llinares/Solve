The parameters are:
===================
- Output prefix
- Box size (Mpc/h)
- grid (1D)
- Omega_0 (omega matter total)
- Nseed
- hsmall
- Model for the power spectrum
    0 -> Formulas from Klypin.
    1 -> Power law.
    2 -> Table with the transfter function given by cmbfast.
	5 -> Table with power spectrum given by grafic.
- Flag for showing the input spectrum
- Flag for showing the output spectrum
- Show sigma8
